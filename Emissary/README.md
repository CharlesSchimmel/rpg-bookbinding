Emissary is formatted for half-letter booklet printing so no additional trimming 
is required. The booklet can be simply staple bound.

# Script Instructions
- Copy your source PDF in this directory.
- Run `./script.sh`

# Duplex - All
For printing on a duplex printer (ie printing both sides of a sheet in the same 
pass). Ex: a fancy printer like at Kinkos, Office Max, Staples, etc.

If you want to print all sheets in color, or all sheets in Black and White, you 
can simply print `./print/Emissary - Duplex - All.pdf`.

If you only want to print the color sheets in color and the rest in B&W (because 
color sheets cost more), print `./print/Emissary - Duplex - Color Pages 
Only.pdf` in color and then `./Emissary - Duplex - BW Pages Only.pdf` in B&W. 
You will need to reorder the sheets (in this case only the first sheet is in 
color).
